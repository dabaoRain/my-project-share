### 组件介绍
   >>> 针对项目中数字输入的限制
   >>> 技术栈
       vue父子组件传值
       vue动态class样式
       正则表达式
       正则表达式的动态变量参数

  #### 组件api 
     numberType===>String类型 integer为正数 float为小数
     decimal====> 
        String | Number类型 默认保留两位小数
        前提在numberType为float为小数时起作用，decimal值为保留小数点后几位小数
     negative==>
        是否可以输入负数 默认是false

    使用实例：
      <Input
        negative
        :decimal="3"
        :numberType="'integer'"
        :width="120"
        :placeholder="'请输入车辆价格'"
        :align="'right'"
        border
        @getInputVal="getInputVal"
     />
  #### 数字输入限制原则
    1.正整数包含0
      （1）只能输入数字0-9
       (2) 首位是0时，后面不能在输入其他数字
    2.正数（正数、负数、0）
      （1）可以输入数字0-9或者-
      （2）首位是0时，后面不能在输入其他数字
      （3）限制输入负号后后面再输入0
      （3）输入数字中只能包含一次负号，而且必须是在数字开头，数字中间不允许输入数字
    3.正浮点数
      （1）只能输入数字0-9或者.
       (2)首位数不能输入.
       (3)首位是0时，后面不能在输入其他数字只可以跟小数点
       (4)数字中只能输入一个小数点
    4.浮点数
      （1）只能输入数字0-9或者.或者-
       (2)首位数不能输入.
       (3)首位是0时，后面不能在输入其他数字只可以跟小数点
       (4)数字中只能输入一个小数点和一个-负号
